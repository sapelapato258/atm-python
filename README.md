# ATM Banking System

A simple ATM simulation built with Python that persists balance data to a file.

## Features
- Withdraw money with balance validation
- Deposit money
- Balance persistence using file handling
- Error handling for invalid inputs

## How to Run
```bash
python p1.py
```

## Technologies Used
- Python 3
- File I/O
- Error handling (try/except)

## What I Learned
- File handling in Python
- While loops for menu systems
- Input validation
- Data persistence
